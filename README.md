# rRituals - Bug Tracker

This repository is used **for reporting bugs and suggesting ideas** related to the **rRituals** plugin.

## 📝 How to report a bug

When creating an issue, please include:

**1. Plugin version**

**2. Server version** (e.g. Paper 1.20.4)

**3. List of other plugins** (if any)

**4. Description of the problem**

**5. Steps to reproduce**

**6. Expected behavior**

**7. Actual behavior**

**8. Logs / errors (if any)**

## Example

```
Version: 1.2.0  
Server: Paper 1.20.4  

Description:
Ritual does not start after mob sacrifice

Steps:
1. Build the ritual structure
2. Kill a mob nearby
3. Nothing happens

Expected:
The ritual should activate

Actual:
Nothing happens
```

## Important

* Check if a similar issue already exists
* Attach logs - reports without logs may be ignored
* The more details you provide, the faster the fix
